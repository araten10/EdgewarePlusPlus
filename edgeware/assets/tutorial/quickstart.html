Quick Start

Want to get Edgeware++ up and running as fast as possible? The default settings are okay, but are very basic. In this section we'll list all the settings you should change before trying Edgeware++ for the first time.

I still highly recommend the other tutorials in this window for new users. If you want to learn more about the program and how everything works, check out the other tutorial tabs first, then come back here for a brief refresher. However, if you want to skip that and get started immediately, then this is for you!

IMPORTANT

  • General → Start. Set Global Panic Key to something you won't accidentally activate, but can also be easily pressed. If you are on windows, you may need to tab out to your desktop to set the hotkey, this is a bug and should hopefully be fixed!
  • General → Start. Ensure that Warn if "Dangerous" Settings Active is turned on, unless you know what you're doing. It should be on by default, but doesn't hurt to double check.
  • Annoyance/Runtime → Wallpaper. Set your panic wallpaper using Auto Import (preferable) or Set Panic Wallpaper.

With these settings on, you now have an easy way to quit Edgeware++ while also reverting your wallpaper to what is currently set on your computer.

Annoyance/Runtime → Popup Types

General Popup Frequency is the most important setting in this tab. To start, I recommend setting it to 8000-10000, although you may want to change it later.

For the types themselves, I recommend prioritizing images and video and give every other popup type a very small percentage:

  • Keep Images at 100%.
  • Turn Videos up to ~10%.
  • Have Subliminals at around 5%.
  • Put Audio at 3%.
  • Prompts and Website are more annoying if they pop up frequently, so try having them at 1-2%. (You can turn website up more if you want)
  • Notifications depends entirely on your OS and how it decides to handle them, you can try them at 3-5%.
